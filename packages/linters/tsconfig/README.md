Shareable TypeScript configuration.

## Installation

```bash
pnpm add -D @alanlu-dev/tsconfig
```

## Usage

Create a `tsconfig.json` file in the root of your project:

```json
{
  "extends": "@alanlu-dev/tsconfig/base.json"
}
```
