Shareable markdownlint configuration.

## Installation

```bash
pnpm add -D @alanlu-dev/markdownlint-config
```

## Usage

Create a `.markdownlint-cli2.cjs` file in the root of your project:

```js
module.exports = require('@alanlu-dev/markdownlint-config')
```
