Shareable ESLint configuration.

## Installation

```bash
pnpm add -D @alanlu-dev/eslint-config-base
```

## Usage

Create a `.eslintrc.cjs` file in the root of your project:

```js
module.exports = {
  root: true,
  extends: ['@alanlu-dev/base'],
}
```
