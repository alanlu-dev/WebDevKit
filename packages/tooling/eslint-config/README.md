Shareable ESLint configuration.

## Installation

```bash
pnpm add -D @alanlu-dev/eslint-config
```

## Usage

Create a `eslint.config.mjs` file in the root of your project:

```js
import config from '@alanlu-dev/eslint-config'

export default config
```
