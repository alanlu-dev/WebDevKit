Shareable lint-staged configuration.

## Installation

```bash
pnpm add -D @alanlu-dev/lint-staged-config
```

## Usage

Create a `.lintstagedrc.cjs` file in the root of your project:

```js
module.exports = require('@alanlu-dev/lint-staged-config')
```
