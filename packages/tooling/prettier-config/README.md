Shareable prettier configuration.

## Installation

```bash
pnpm add -D @alanlu-dev/prettier-config
```

## Usage

Create a `.prettierrc.cjs` file in the root of your project:

```js
module.exports = require('@alanlu-dev/prettier-config')
```
