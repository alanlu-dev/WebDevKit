Shareable TypeScript types.

## Installation

```bash
pnpm add -D @alanlu-dev/types
```

## Usage

Create a `tsconfig.json` file in the root of your project:

```json
{
  "compilerOptions": {
    "types": ["@alanlu-dev/types/globals"]
  }
}
```
